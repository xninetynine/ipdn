Introduction to Programming

Stage 0: Getting Started with HTML

What is HTML?

HTML stands for Hyper Text Markup Language and consists of text content, mark up, references to other documents and links to other pages. HTML uses elements to structure the webpage.

What is an element? Elements are what make up up a html document, they generally are enclosed within a pair of tags. Tags have an opening and a closing tag.
  • <opening tag> content </closingtag>
Generally tags come in pairs, but certain tags such as < br > do not need a closing tag and are considered void tags. Also some elements accept additional values called "attributes".

Types of Elements

Inline Elements:
  • <a> content </a> defines hyperlinkes
  • <b> content </b> defines bolded text
  • <br> defines single line break
  • <em> content </em> defines emphasized text
  • <img> defines an image
  • <span> defines a section in a document
  • <strong> defines important text
Block Element
  • <div> defines a section in a document; is a container
  • <form> defines a section in a document
  • <p> defines a section in a document; is a container

Stage 1: Make a Web Page

Building Blocks: HTML, CSS, JavaScript

HTML is the foundation component of a webpage, similar to the foundation or wall placement that determines what rooms go where

CSS is the style component of a webpage, similar to deciding wall color or carpet type, this is the aesthetic part of design

JavaScript are the interactive components such as a TV remote, or passcode entry lock

Construction: Developer Tools

Remember, everything is built in blocks! Boxes break down from large to small.

Document Object Model(DOM) builds the webpage based on html elements. Which consists of tags, attributes and values.
